Architecture

Infrastructure complète
en un coup d'œil

CloudBot repose sur une architecture microservices en couches, conçue pour la résilience et l'observabilité. Chaque composant est isolé dans son propre conteneur Docker, communiquant via un réseau interne sécurisé.

Le trafic utilisateur entre par Nginx, qui le route vers Flask. Flask dialogue avec Ollama pour l'IA et MySQL pour la persistance des conversations.

  • Nginx comme point d'entrée unique et sécurisé
  • Flask API REST avec métriques Prometheus intégrées
  • Ollama TinyLlama pour le traitement LLM local
  • MySQL 8.0 avec volumes Docker persistants
  • Stack Prometheus + Grafana pour l'observabilité
  • UFW + réseau Docker isolé pour la sécurité
